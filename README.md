# FiveM: Clothing Shops addon
> The best clothing shop addon for FiveM

# Caution: WORK IN PROGRESS

## Requirements
- EssentialMode 2.X
- MySQL

## Features
- Fast loading
- Load skin on spawn
- Buy new clothes (and it updates in the DBMS)
- Blips on the map
- Marker in shops
- Elegant menu
- A lot of accessories
